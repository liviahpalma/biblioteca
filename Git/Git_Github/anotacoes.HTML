GIT e GITHUB - CONCEITOS *GIT - é um sistema de versionamento, controle de versão, “máquina do tempo”, pode não ser a melhor escolha para todos, mas é uma escolha para diversas possibilidades de controle. É um sistema de controle de versão de arquivos, onde diversas pessoas poderão contribuir ao mesmo tempo para um projeto hospedado, editando, criando sem riscos de serem sobrescritos, padronizando uma versão para que possam haver alterações sem perigo, havendo diversos “snapshots” do seu projeto, tendo como função, criar, compartilhar, versionar projetos.
